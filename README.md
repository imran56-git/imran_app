# Find Your Best Teacher Today

A full-featured Flutter app connecting students directly with teachers without any third-party agency.

## Features

- Firebase Google Sign-In
- Teacher & Student Profiles
- Subject-wise Teacher Search & Filter
- Google Map Integration with Teacher Markers
- One-to-One Chat with:
  - Emoji Support
  - Image Sharing
  - Voice Message
  - Typing/Online/Last Seen
- FCM Push Notifications
